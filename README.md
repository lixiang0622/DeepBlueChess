# DeepBlueChess ♟️

深蓝国际象棋社比赛管理系统与资料库。

## 项目简介

本项目包含深蓝国际象棋社的**瑞士轮赛事管理系统**（Web 版与桌面版）、**棋谱存档**、**摊位题库**以及**社团资料**。

## 目录结构

```
DeepBlueChess/
├── src/                    # 源代码
│   ├── app.py              # Streamlit Web 版瑞士轮比赛管理系统
│   ├── swiss_round.py      # Tkinter 桌面版瑞士轮比赛管理系统
│   └── ico.py              # PNG 转 ICO 图标工具
├── assets/                 # 静态资源
│   └── icon.ico
├── games/                  # 棋谱存档（PGN 格式）
│   ├── 线上锦标赛/
│   ├── 线下校内比赛/
│   └── 校际比赛/
├── exam/                   # 摊位题库
├── build_config/           # 构建配置
├── legacy/                 # 历史版本
└── 社团资料/               # 社团文档（学习资料、电子书、会议纪要、活动记录）
```

## 瑞士轮比赛管理系统

支持 Web 端（Streamlit）和桌面端（Tkinter）两种运行方式，功能包括：

- **选手报名**：手动添加或 Excel 批量导入，支持学号、姓名、队伍、性别信息
- **瑞士轮配对**：基于积分、对手分（Sonneborn-Berger）的自动配对引擎，支持颜色平衡、同队规避、递归回溯求解
- **比分录入**：白胜/黑胜/和棋/弃权等多种结果快速录入
- **实时排名**：积分 → 小分 → 胜局数三级排序
- **裁判操作**：手动加减分、选手退赛/恢复资格
- **报表导出**：每轮对阵表/结果图 + 最终汇总 PDF（含排名表与选手个人卡片）
- **状态保存**：比赛进度 JSON 存档，支持中断后恢复
- **中英双语**：Streamlit 版支持中文/英文界面切换

### Web 版运行

```bash
pip install streamlit pandas matplotlib
streamlit run src/app.py
```

### 桌面版运行

```bash
pip install matplotlib
python src/swiss_round.py
```

## 棋谱存档

`games/` 目录下按赛事类型分类存放 `.pgn` 格式棋谱：

- **线上锦标赛**：深蓝之夜系列、跨年首战、山巅之战等
- **线下校内比赛**：Lichess 校内对局
- **校际比赛**：预留

## 题库

`exam/` 目录包含国际象棋摊位题库（PDF + LaTeX 源码），用于社团摊位活动。

## 社团资料

预留目录，用于存放社团的学习资料、电子书资源、会议纪要和活动记录。
